# 风控阈值与评分标准 (Risk Matrix)

## 1. 阈值定义
| 指标 | 默认值 | 说明 | 数据依据 |
| :--- | :--- | :--- | :--- |
| Window | 20 | 审计窗口 | - |
| Min Amount | **2000** | 对应 **200万元**。 | 市场 10% 分位约为 974 (100万)，取双倍安全边际。 |
| Max Volatility | **0.02** | 日收益率标准差 **2%**。 | 市场 95% 分位为 1.77%。超过 2% 属极端高波。 |

## 2. 评分与决策
系统采用 **扣分制** 计算 `risk_score` (0-100)：
- **0-49分**: **PASS** (低风险，允许交易)
- **50-100分**: **REJECT** (高风险，建议剔除)

### 扣分细则
- **流动性枯竭**: +60分 (一票否决)
- **波动率异常**: +30分 (警示)
- **数据严重缺失**: +100分 (直接拒绝)
